# ToolsPack: collection of useful codes

[![Run all tests](https://github.com/duongphuhiep/ToolsPack.NetCore/actions/workflows/ci.yml/badge.svg)](https://github.com/duongphuhiep/ToolsPack.NetCore/actions/workflows/ci.yml)

My collection of many small useful code-snippets .net, wrapping in reusable [library on NuGet](https://www.nuget.org/profiles/duongphuhiep) with unobtrusive dependencies.

This repository won't got much updates because the codes are very stable through years of battle-testing on production hence nearly Zero-maintenance is needed.

The nugets usually targets .NETStandard and the 2 last versions of .NET.

Navigate to the `README.md` in each packages for more information.

* [ToolsPack.String](./src/ToolsPack.String/README.md)
* [ToolsPack.Logging](./src/ToolsPack.Logging/README.md)
* [ToolsPack.Msql](./src/ToolsPack.Msql/README.md)
* [ToolsPack.Thread](./src/ToolsPack.Thread/README.md)
